**🎓 Smart Classroom Monitoring System**
This project is an automated Smart Classroom Monitoring System built using Arduino Uno 🛠️. It is designed to monitor environmental conditions and human presence to optimize energy usage and ensure a comfortable classroom environment 🌍.

🌟 Key Features
🌡️ Temperature Monitoring: Uses a TMP36 sensor to track room temperature and automatically trigger a cooling fan 🍃.

💡 Light Intensity Control: Uses an LDR (Light Dependent Resistor) to monitor ambient light levels and control lighting automatically ☀️.

🚶 Motion Detection: Uses a PIR sensor to track human presence inside the classroom 🔒.

🖥️ Real-time Display: A 16x2 LCD screen displays real-time data, including temperature, light percentage, motion status, and occupancy count 📊.

🔌 Components Used
Microcontroller: Arduino Uno R3 🤖

Sensors:

TMP36 Temperature Sensor 🌡️

LDR (Light Dependent Resistor) ☀️

PIR Motion Sensor 🏃

Output Devices: 16x2 LCD Display 🖥️, LEDs 💡, Buzzer 🚨

Others: Breadboard, resistors, and jumper wires ⚡

⚙️ How It Works
The system continuously reads data from the TMP36, LDR, and PIR sensors 🔍.

It processes this data using specific logic to control peripheral devices like fans, lights, and buzzers ⚙️.

All relevant status information is updated on the LCD screen for easy monitoring 👀.
